# Academic Website: aditya08.github.io

This repository builds an academic website using [Hugo](https://github.com/gohugoio/hugo).
It is based on the modified [PaperMod theme](https://github.com/adityatelange/hugo-PaperMod) by [Pascal Michaillat](https://pascalmichaillat.org/hugo-website/).
The website is hosted on [GitHub Pages](https://docs.github.com/en/pages/getting-started-with-github-pages/about-github-pages).
